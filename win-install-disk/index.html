前情提要

今天给堂弟的电脑重装系统时，遇到了以往USB安装系统经常出现的问题，“Windows无法安装到这个磁盘。选中的磁盘采用GPT分区形式”。之前在CSDN的博客上有记录相关解决方法，但因CSDN博客打开繁琐，点击层级太多，因此转移至此，以作备份。
